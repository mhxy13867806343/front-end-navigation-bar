## 1. 设计支柱 (Design Pillars)
1. **冻结即掌控** — 玩家不是"逃避"敌人，而是把敌人变成可被利用的武器（雪球）。从守势翻转为主动是核心爽点。
2. **连锁满足感** — 滚一颗雪球沿途撞杀一排敌人，combo 分数指数增长，是"再来一次"的主引擎。
3. **垂直推进** — 单屏多层平台，靠"踩雪球弹跳"上高台，清场后右上角门亮起过关，节奏明快不拖沓。
4. **双人同屏** — 蓝(Nick)/红(Tom) 各自 3 命，互为友军（不互伤），共享清场目标，合作清关或比拼分数。

## 2. 核心循环 (Core Loop)
### Moment-to-Moment (0–30s)
- **Action**: 移动 + 朝敌喷雪（按住射击）
- **Feedback**: 雪雾粒子 + 敌人逐渐变白（冻结度进度条）+ 冻结"咔"音效
- **Reward**: 敌人变雪球 → 推/滚/踩 的支配感

### Session Loop (单关 1–3 分钟)
- **Goal**: 清空本屏所有敌人 → 右上角门激活 → 走入过关
- **Tension**: 敌人接触掉命；雪球融化会复活（约 4s `[PLACEHOLDER]`）；滚动雪球也能砸伤自己
- **Resolution**: 过关 → 分数结算 → 下一关；命数耗尽 → Game Over

### Long-Term Loop (3 关纵切片)
- **Progression**: 关 1～2 普通层（难度/敌人数递增）→ 关 3 Boss
- **Retention Hook**: 分数榜（localStorage 最高分）+ combo 连杀追求的"完美清场"

## 3. 机制规格 (Mechanic Specs)

### M1. 喷雪冻结 (Snow Spray)
- **Purpose**: 把敌人转化为可控雪球，是全部玩法的起点
- **Player Fantasy**: "我把威胁冻住了，现在它是我的弹药"
- **Input**: 按住 射击键（P1=J, P2='.'）
- **Output**: 朝 facing 发射雪粒子；命中敌人累加 `freeze`；`freeze>=FREEZE_FULL` → 敌人转为雪球实体
- **Success**: 敌人在 ~0.55s 内被冻成球（手感爽快）
- **Failure**: 喷射时移动减速，被敌接触掉命
- **Tuning Levers**: `FREEZE_RATE`(每帧+3)、`SPRAY_SPEED`、喷射时 `MOVE_MULT`(0.5)
- **Dependencies**: 敌人状态机、雪球生成、道具生成

### M2. 推 / 滚雪球 (Push / Roll)
- **Purpose**: 利用雪球清场与连杀
- **Input**: 朝雪球方向移动接触
- **Output**: 雪球获得 `vx`（带摩擦衰减）；滚动中撞敌 → 击杀 + combo++
- **Success**: 一颗球滚穿 3+ 敌，combo 跳字
- **Failure**: 撞墙/屏边停下（开始融化）；或滚出屏外销毁（不复活）
- **Edge Cases**:
  - 雪球滚动中 `melt` 计时暂停（只有静止才融化）
  - 滚动雪球撞到玩家 → 玩家受伤（含自己推的，加 0.3s 推出缓冲）
- **Tuning Levers**: `PUSH_SPEED`(6)、`FRICTION`(0.92)、`ROLL_KILL_MIN_VX`(2.5)
- **Dependencies**: 敌人、combo/分数、玩家受伤

### M3. 踩雪球弹跳 (Stomp Bounce)
- **Purpose**: 垂直移动手段，上高台的核心
- **Input**: 从上方落体踩中雪球顶
- **Output**: 玩家获大上跳速度 `BOUNCE`(-13)
- **Success**: 借雪球弹上原本跳不到的平台
- **Edge Cases**: 雪球正在滚动时被踩 → 仍可弹起，雪球保留 `vx`
- **Dependencies**: 玩家跳跃、平台高度差设计

### M4. 道具 (Potions) — 敌变球时 70% 概率生成
- 红=加速移动 ｜ 黄=冻结更快(×2) ｜ 蓝=雪雾范围更大 ｜ 绿=加分+补时
- **Duration**: 8s `[PLACEHOLDER]`；漂浮于雪球上方，玩家碰到获得
- **Buff 各自玩家独立生效（双人）**

### M5. Boss（关 3）
- 大号敌，HP=10 `[PLACEHOLDER]`；可被冻；冻成球态时玩家"踩"其顶 → 扣 1 HP 并解冻
- Boss 周期发射雪弹（命中玩家掉命），逼迫玩家"冻→踩→冻→踩"循环

## 5. 崩溃定义 (Broken State — playtest 前先锁定)
- **太难**: 平均单关死亡 >2 命 或 80% 玩家卡关 3 分钟不过 → 降 `FREEZE_FULL`/敌速
- **太易**: combo 随便上 10+ 且 0 死亡 → 增敌数 / 缩短融化时间
- **卡死**: 雪球全部融化复活导致永远清不了场 → 必须允许"推下屏"销毁
- **双人失衡**: 一人推球误伤另一人导致互相抱怨 → 友军雪球伤己缓冲 0.3s，且不加 combo 给受害者

## 6. 关卡 (3 关纵切片)
- **几何约束**: 普通跳跃最大高度约 100px；门必须能通过普通跳跃到达，雪球弹跳只作为 shortcut/爽点，不能是必需。
- **关 1**: 地面 → 左侧 4 级楼梯平台（落差~60px）→ 右上门，5 小怪 — 教学：冻/推/踩。
- **关 2**: 地面 → 4 级楼梯平台 → 右上门，6 小怪，1 必出道具 — 强化连杀。
- **关 3**: Boss 关，左右两侧楼梯上到中央平台，Boss 被限制在中央平台不跌落，1 Boss + 2 小怪 — 终战。
- 通关关 3 → WIN，记录最高分
