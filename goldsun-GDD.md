# 黄金太阳 · 封印篇 — Game Design Document (v0.1)

> 复刻对象：南晶科技 FC《黄金太阳》（GBA《黄金太阳：开启的封印》同人移植，16Mb 回合制 RPG）
> 交付形态：单文件 HTML5 Canvas 纵切片（`goldsun.html`，零依赖 + WebAudio 合成音效）
> 设计：GameDesigner — 核心幻想 = **回合制元素战斗 + 精神力 + 精灵召唤 + 救世冒险**

---

## 0. 范围声明（重要）

原作是完整多章节 JRPG（世界地图、城镇、迷宫、随机遭遇、精神力解谜、精灵收集召唤、剧情）。一次性 1:1 复刻不现实，本切片**聚焦最被记住的核心**——**回合制元素战斗系统**，外加一层轻量大地图把战斗串起来。完整的 RPG 外壳（多章节/城镇/迷宫解谜/存档）见文末《扩展路线图》。

---

## 1. 设计支柱（Design Pillars）

1. **元素即策略** — 火→风→土→水→火 的克制环必须让玩家"看得见收益"，每一发法术都有属性博弈。
2. **精灵召唤是爽点** — 收服精灵后，战斗中可召唤神兽打出全体重创，是冒险感与成长回报的高潮。
3. **轻量但完整的一轮冒险** — 走地图 → 收精灵 → 打小怪练级 → 头目决战，5 分钟内可完整体验"开始到通关"。
4. **零门槛可玩** — 键盘 + 触屏双输入，菜单可点可键，不需要读说明书。

---

## 2. 核心循环（Core Loop）

### Moment-to-Moment (0–10 秒)
- **Action**: 踩草地随机遭遇 / 头目门前开战
- **Feedback**: 切到战斗场景、BGM 闪烁、指令菜单弹出
- **Reward**: 一次出招命中跳伤害数字、属性克制时高伤反馈

### Session Loop (5–10 分钟)
- **Goal**: 收服精灵·索尔 → 清小怪攒 EXP/金币 → 击败头目·萨丢罗斯
- **Tension**: MP 有限（精神力/召唤都要耗蓝）、头目每 3 回合放大招
- **Resolution**: 胜利 → 金币/经验入账、可升级；失败 → GAMEOVER 重开

### Long-Term Loop（扩展后）
- 多章节地图、城镇补给、迷宫精神力解谜、精灵图鉴收集、对战模式

## 3. 机制规格（Mechanic Specs）

### 机制 A：回合制战斗（核心）
- **Purpose**: 提供"读敌属性 → 选克制手段 → 看数值反馈"的策略循环
- **Player Fantasy**: 我是操控元素的炼金术士
- **Input**: 指令菜单（攻击 / 精神力 / 精灵 / 防御 / 逃跑）
- **Output**: 伤害/治疗/状态变更，敌我 HP/MP 变化
- **Success**: 玩家能用属性克制稳定压低敌人血量
- **Failure**: 乱用 MP 导致头目战没蓝放大招 → 崩盘
- **Edge Cases**:
  - 玩家全灭 → GAMEOVER
  - 敌人全灭 → 结算 EXP/金币/升级
  - 头目战不可逃跑
- **Tuning Levers**: `ENCOUNTER_RATE`、各角色 ATK/DEF/SPD、法术 `power`/`cost`、属性倍率 `EL_MULT_STRONG/WEAK`
- **Dependencies**: 精灵系统（召唤）、升级系统、地图遭遇

### 机制 B：精神力（Psyenergy / 法术）
- **Purpose**: 让属性克制落到实处；提供输出与治疗的分工
- **罗宾**: 盖亚之刃(土,4MP,24) / 冰冻(水,5MP,20) / 治愈(水,6MP,30 治疗)
- **杰拉德**: 烈焰(火,4MP,26) / 爆炎(火,8MP,44)
- **Output**: 消耗 MP，造成属性伤害或治疗
- **Edge**: MP 不足时菜单提示"精神力不足"，不可释放

### 机制 C：精灵召唤（Djinni Summon）
- **Purpose**: 成长回报高潮、清场手段
- **Player Fantasy**: 我养的精灵替我翻盘
- **Input**: 战斗中「精灵」指令（需已收服且本场未用过）
- **Output**: 对全体敌人造成 `DJINNI_POWER`(52) × 属性倍率 伤害；本场战斗只能召唤 1 次
- **Failure**: 未收服精灵 / 已用过 → 提示不可用
- **Tuning**: `DJINNI_POWER`、是否允许每场多次（当前 1 次，符合"珍贵资源"设计）

### 机制 D：属性克制
- 环：火→风→土→水→火
- 克制 1.5×、被克 0.75×、其余 1×（见 `BEATS` / `elMult`）
- 设计意图：让"选对法术"比"无脑攻击"收益明显，制造决策

### 机制 E：升级 / 经济
- `EXP_BASE = 50`：升级所需 = `level × 50`
- 升级：HP+12 / MP+4 / ATK+3 / DEF+2 / SPD+1，回满
- 金币累计，最高分存 `localStorage('goldsun_hi')`

## 4. 平衡数值表（带理由 · `[PLACEHOLDER]` 待实测）

| 变量 | 基值 | 说明 |
|---|---|---|
| MOVE_SPEED | 165 | 大地图移动速度，体感顺滑 |
| ENCOUNTER_RATE | 0.045 | 每步踩草地触发概率；太高烦、太低无聊 `[PLACEHOLDER]` |
| FLEE_RATE | 0.6 | 普通战逃跑成功率 |
| EL_MULT_STRONG | 1.5 | 克制倍率；试 1.4~1.6 |
| EL_MULT_WEAK | 0.75 | 被克倍率 |
| EXP_BASE | 50 | 升级曲线斜率 `[PLACEHOLDER]` |
| DJINNI_POWER | 52 | 召唤单体积伤；应≈小怪 1.5~2 条命 |
| 罗宾 HP/MP | 62/30 | 偏脆皮法师，靠治愈自保 |
| 杰拉德 HP/MP | 74/22 | 血厚近战火法 |
| 哥布林 HP | 32 | 开胃怪 |
| 石像鬼 HP | 46 | 中怪 |
| 萨丢罗斯 HP | 230 | 头目；约需 4~6 轮集火 |

**"崩溃"定义（broken）**：
- 头目可在玩家出 1 招前被召唤秒杀 → 召唤过强
- 普通战 MP 永远用不完 → 精神力无成本
- 踩草地 10 步内必遇敌 → 探索被惩罚

## 6. 扩展路线图（把切片补成完整 RPG）

1. **大地图扩充**：多屏滚动 + 城镇（补给/存档点）+ 多区域（草原/洞窟/火山）
2. **精神力解谜**：在地图上用"移动/冰冻/旋风"推石、冻水、清藤开路（原作招牌）
3. **精灵系统**：4 元素多只精灵，可分配给队员改变属性/召唤兽
4. **剧情章节**：3 章（对应原作宇宙/西部/海底式结构），过场对话
5. **对战模式**：读取存档 3 人组队，竞技场 PvP
6. **存档**：localStorage 持久化队伍等级/精灵/进度
